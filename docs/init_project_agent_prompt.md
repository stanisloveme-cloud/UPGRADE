# UPGRADE CRM: Agent Initialization Prompt

**Назначение:** Используйте этот текст как системный промпт (или скопируйте его в первое сообщение) при создании новой сессии с любым ИИ-агентом (разработчиком, архитектором, тестировщиком), чтобы он мгновенно погрузился в контекст проекта и не совершал архитектурных ошибок.

---
## СКОПИРУЙТЕ ТЕКСТ НИЖЕ И ОТПРАВЬТЕ АГЕНТУ:
---

Привет, агент! Добро пожаловать в проект **UPGRADE CRM**. 
Твоя первая и самая главная задача перед тем, как писать код или отвечать на мои вопросы — **провести инициализацию и загрузить системный контекст**.

1. Прямо сейчас, до любых других действий, выполни внутреннюю команду-воркфлоу: `/init_agent`.
2. Внимательно прочти файл `d:\UPGRADE\.agent\workflows\init_agent.md`, который эта команда вызовет.
3. Обязательно выполни все шаги из первого пункта воркфлоу — прочитай `d:\UPGRADE\.antigravityrules` (это критически важный свод правил, за нарушение которого агент получает штраф).
4. Убедись, что ты понимаешь запрет на использование терминального `git` и обязанность использовать ТОЛЬКО интеграцию MCP GitHub для коммитов.
5. Прочитай файл `d:\UPGRADE\docs\PROJECT_STATUS.md`, чтобы понимать, на какой стадии мы находимся и какие изменения были сделаны до тебя. Запомни, что по окончании работы ты обязан будешь дописать туда свой Change Log.

Как только ты прочитаешь всю документацию, коротко отчитайся мне: 
"Я инициализирован, правила `.antigravityrules` и статус проекта прочитаны. MCP GitHub активен. Контекст UPGRADE CRM загружен. Жду ваших задач!"
