CMS  для работы с деловой программой и спикерами.

Как выглядит система в текущем виде

Пользовательский интерфейс: 
 https://sales.upgradecrm.ru/dashboard vladislav.shirobokov@gmail.com vladislav456
Разделы:
День конференции
поле для редактирования продолжительности сессий
названия конференций и залов на конкретной площадке (Маркетинг в зале 1)
вопросы деловой программы: название сессии / краткое описание сессии / инфо о модераторе / вопросы деловой программы 
раздел спикеры:  Имя, Фамилия, Должность, Компания, Фото / контакы (телефон, телеграм, мейл) / контактное лицо (Имя, Фамилия, Должность, Компания, телефон, телеграм, мейл) / BIO (чем спикер полезен)
статусы спикеров:  добавлен в лонг лист для приглашения (не виден в публичной деловой программе на сайте) / подтвержден (виден в деловой программе) / назначен брифинг / пришле на брифинг / отправлена памятку по участию /  есть презентация или нет / прислал презентацию или нет пришел не площадку
Спикеров заносим лонглистом, подтягиваем контактные данные из записной книжки (база контактов)


Клиентский интерфейс https://spring.upgrade.st/program

Разделы деловой программы: 
дент конференции (навигация между днями, переключение)
слева = тематики конференций и навигация
средняя колонка  = подробная деловая программа
справа = подробная информация о спикере

Разделы о  спикерах:
иконостас = фото, имя, фамилия, должность, компания, BIO (окно всплывающее под i) https://spring.upgrade.st/speakers
список спикеров  без фото  https://spring.upgrade.st/speakers-list 
Очередность показа спикеров можно произвольно менять (выше / ниже в списке в зависимости от статуса)
