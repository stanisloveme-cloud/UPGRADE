# Инструкция по интеграции расписания UPGRADE CRM в Tilda

Данный документ описывает процесс встраивания динамического расписания конференций, мероприятий и биографий спикеров из системы UPGRADE CRM на любой сайт, созданный в конструкторе Tilda.

## 1. Подготовка

Прежде чем начать интеграцию, убедитесь, что:
1. Вы знаете **ID мероприятия** (Event ID) в системе UPGRADE CRM (например, `76`).
2. У вас есть доступ к редактированию страницы в Tilda.
3. Процесс модерации контента завершен на стороне CRM (расписание составлено, спикеры и их биографии подтверждены с флагом "Отображать на сайте").

## 2. Пошаговая интеграция в Tilda

Отображение программы строится на использовании стандартного блока Tilda для вставки HTML-кода — **T123 (HTML-код)** или через вставку кода внутри **Zero Block**.

### Шаг 1: Добавление HTML-блока
1. Откройте нужную страницу в конструкторе Tilda.
2. Нажмите круглый плюс **"+"** (Добавить блок) в том месте, где должно выводиться расписание.
3. В боковом меню выберите раздел **"Другое"**, затем кликните на блок **T123 (HTML-код)**.

### Шаг 2: Вставка сниппета
Сниппет представляет собой единый блок HTML, включающий стили (`<style>`) и логику на JavaScript (`<script>`). 

1. Актуальную версию кода интеграции (сниппета) разработчики поддерживают в файле инфраструктуры: `scripts/tilda-integration-snippet.html`.
2. Скопируйте **весь** исходный код из данного файла.
3. В настройках добавленного блока T123 на странице Tilda нажмите кнопку **"Контент"**.
4. Вставьте скопированный код в текстовое поле.

### Шаг 3: Настройка ID мероприятия (event ID)
В скопированном коде внизу, внутри тега `<script>`, найдите строку инициализации:

```javascript
const EVENT_ID = window.crmEventId || 76; 
```

Вы ক্ষমতায় напрямую заменить `76` на тот ID, расписание которого необходимо вывести. Например, для вывода программы E-COM СФЕРА (ID: 2):

```javascript
const EVENT_ID = window.crmEventId || 2; 
```

### Шаг 4: Публикация страницы
1. Нажмите "Сохранить и закрыть" в меню редактирования контента HTML-блока.
2. Опубликуйте страницу Tilda.
3. Перейдите по ссылке опубликованной страницы. Расписание должно подгрузиться автоматически с бэкенда платформы `devupgrade.space4u.ru` (и далее с продакшена `upgrade.st`).

## 3. Возможности и визуализация

Интеграция поддерживает следующий функционал из коробки:
- **Адаптивная верстка:** Корректное отображение на ПК, планшетах и мобильных телефонах.
- **Вкладки по дням:** Если мероприятие идет несколько дней, скрипт автоматически генерирует интерфейс табов (вкладок) с датами проведения.
- **Тезисы и вопросы:** Автоматический вывод связанных с сессией вопросов к обсуждению под описанием сессии.
- **Модальное окно спикера:** При клике на "Подробнее о спикере" открывается всплывающее окно (pop-up) с большим фото (с применением стильного ЧБ-фильтра `grayscale(100%)`), должностью, названием компании и полной биографией.
- **Интеграция якорей:** Кнопка "Участвовать" настроена на автоматическую прокрутку к форме регистрации по якорю `#registration`.

## 4. Локальное тестирование для дизайнеров UX

Для удобной отладки внешнего вида и внесения визуальных изменений в стили изолированно от Tilda:

Откройте в браузере предоставленную тестовую среду (StandAlone-версия):
`https://devupgrade.space4you.ru/test-tilda-standalone.html`

- В верхней части страницы находится селектор выбора мероприятия. 
- Изменив `eventId` в URL (например, `?eventId=42`), вы можете проверить, как отображаются различные мероприятия.
- Чтобы поменять стили, вы можете редактировать инлайн-блок `<style>` прямо внутри сниппета `scripts/tilda-integration-snippet.html` в кодовой базе проекта, коммитить изменения и обновлять локальную страницу — это гарантирует, что дизайн, протестированный вами в StandAlone, 1:1 переедет на боевой сайт Tilda.
