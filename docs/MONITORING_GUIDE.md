# Руководство по мониторингу UPGRADE CRM

В проекте реализована двухуровневая система мониторинга. Ниже описано, как получить к ней доступ и как ей пользоваться.

## 1. Встроенный дашборд состояния системы (Внутри CRM)

Этот дашборд предназначен для быстрого понимания того, жива ли система сейчас. Он встроен прямо в интерфейс UPGRADE CRM.

**Как зайти:**
1. Зайдите в веб-интерфейс UPGRADE CRM под пользователем с правами **SuperAdmin**.
2. Обратите внимание на левое боковое меню навигации. В самом низу, в блоке административных ссылок, появился новый пункт **"Статус системы"**.
3. Кликните на него. Откроется дашборд с виджетами:
   - **Статус приложения:** Общая оценка (Работает / Ошибка).
   - **База данных (PostgreSQL):** Успешен ли коннект Prisma к БД.
   - **Загрузка памяти (Heap / RSS):** Индикаторы потребления оперативной памяти Node.js процессом. Если память утекает, статус загорится красным.
   - **Свободное место на диске:** Если диск сервера заполнен более чем на 90%, статус изменится на красный $\rightarrow$ сигнал, что пора чистить логи или бэкапы.
   - **Доступность Frontend:** Пинг самого веб-сервера.

*Данные на этой странице обновляются автоматически каждые 15 секунд. Вы также можете нажать кнопку "Обновить" вручную.*

## 2. Глубокий мониторинг инфраструктуры (Grafana)

Для детального анализа (графики нагрузки процессора, история потребления памяти, нагрузка по каждому конкретному Docker-контейнеру) мы используем стек **Prometheus + Grafana**. Этот стек запускается автоматически на DevStand сервере вместе с основным приложением.

**Как зайти:**
1. Откройте в браузере адрес: `http://devupgrade.space4you.ru:3001` (или http://IP_сервера:3001, если домен еще не привязан к этому порту/настраивается прокси).
2. На экране авторизации введите логин и пароль по умолчанию:
   - **Логин:** `admin`
   - **Пароль:** `admin`
3. Сразу после входа Grafana попросит вас придумать новый, безопасный пароль.

### Как настроить дашборды в Grafana
Grafana уже собирает все данные (через Prometheus, Node Exporter и cAdvisor), но для красоты их нужно вывести на графики. Самый простой способ — импортировать готовые красивые дашборды:

1. В левом меню Grafana нажмите на значок **"+"** и выберите **"Import"**.
2. В поле *"Import via grafana.com"* введите ID нужного дашборда и нажмите **Load**:
   - 💡 **ID: 1860** — Отличный дашборд ("Node Exporter Full") для показа нагрузки на сам Linux-сервер (CPU, RAM, Диск, Сеть).
   - 💡 **ID: 14282** — Дашборд для мониторинга конкретных Docker-контейнеров (cAdvisor).
   - 💡 **ID: 19688** — Дашборд метрик для NestJS (собирает статистику по HTTP-запросам бэкенда).
3. При импорте в самом низу в поле *Prometheus* выберите единственный доступный источник (наш локальный Prometheus) и нажмите **Import**.

Готово! Теперь у вас есть красивые и профессиональные графики всего, что происходит под капотом UPGRADE CRM.
