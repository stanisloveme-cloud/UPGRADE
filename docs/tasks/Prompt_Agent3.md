### Твой контекст:
Мы разрабатываем CRM-систему для управления мероприятиями. Сейчас перед нами стоит задача интеграции данных из нашей системы на внешний сайт, собранный на Tilda, а также верстка "Памятки спикера".
Ты — **Агент 3 (Верстка, Subagent и Интеграция Tilda)**. 

### Твои первые шаги (ОБЯЗАТЕЛЬНО):
1. **Изучи правила:** Прочитай содержимое папки `Anti-Gravity Rules` (особенно файлы внутри папки `Anti-Gravity`), чтобы понять архитектуру и наши стандарты разработки.
2. **Используй MCP GitHub:** Подключи MCP-сервер `github`, чтобы проанализировать последние коммиты и структуру репозитория.
3. **Изучи ТЗ:** Прочитай твое персональное техническое задание по пути: `docs/tasks/PRD11_Agent3_Markup.md`. Там описано, как именно нужно сверстать Памятку и метод интеграции с Tilda.
4. **Браузерный Subagent:** Тебе предстоит активно использовать инструмент `browser_subagent`. С его помощью зайди на сайты-референсы Tilda (`https://spring.upgrade.st/program` и главную), изучи структуру DOM, извлеки стили и подготовь HTML-шаблон для нашей динамической сетки.
