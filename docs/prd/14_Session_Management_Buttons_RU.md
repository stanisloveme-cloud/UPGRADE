# RD 14: Кнопки управления сессией (Экспорт и Печать)

## 1. Контекст
В предыдущей версии редактора программы (Legacy CRM) в блоке управления сессией присутствовал важный набор из трёх кликабельных кнопок. В текущей версии (V2 на React/AntDesign) этот функционал отсутствует и подлежит немедленной реализации. Данные кнопки предназначены для быстрой подготовки модераторов и генерации физических материалов (табличек) на площадке.

*Примечание: При аудите предыдущих RD-файлов (00-13) данный функционал подробно не описывался. Запрос является новым актуализированным требованием.*

## 2. Требования к UI
- Кнопки должны располагаться в "блоке управления сессией" (целесообразно разместить их в `SessionDrawer` рядом с кнопкой удаления или в шапке (Extra) модального окна). Кнопки видны только для уже созданных сессий (наличие ID).
- Интерфейс должен быть интуитивно понятным, сопровождаться иконками.
- Кроссбраузерная совместимость (особенно методы вызова окна `window.print()` и `navigator.clipboard`).

## 3. Функциональные требования

### 3.1 Кнопка «Копировать текст»
**Сценарий:** При нажатии данные копируются в системный буфер обмена. Пользователю показывается Toast-уведомление об успехе.
**Состав данных (Plain text):**
- Описание сессии (при наличии)
- Вопросы сессии (нумерованным или маркированным списком)
- Прочие текстовые поля
*(Строго: без списка спикеров)*

### 3.2 Кнопка «Печать для модератора»
**Сценарий:** Открытие системного диалогового окна печати браузера (с возможностью сохранения в PDF встроенными средствами ОС).
**Содержимое документа:**
- Название сессии (Крупный заголовок)
- Дата и время, зал проведения
- Модератор(ы) сессии
- Подтверждённые спикеры (с отметкой о наличии презентации)
- Подробное описание сессии
- Вопросы для обсуждения
**Требования к верстке:**
- Удобочитаемый формат (шрифт не менее 12-14pt, контрастность), отсутствие лишних элементов UI портала (кнопок, меню). Стилизация через `@media print`.

### 3.3 Кнопка «Печать табличек»
**Сценарий:** Одновременная генерация печатной формы табличек для всех подтвержденных спикеров сессии (включая модератора).
**Формат и верстка:**
- Размер: A5 (148 x 210 мм).
- Ориентация: Альбомная (горизонтальная), оптимизирована для складывания "домиком". Каждая табличка должна располагаться на отдельной логической печатной странице (через CSS `page-break-after: always`).
**Содержимое таблички:**
- ФИО спикера (максимально крупно по центру)
- Компания / должность (отдельным блоком ниже)
