- Используем Orval для генерации хуков на клиенте `client/src/api-generated/`.
- Старый код (`axios`) продолжает работать. Миграция идет покомпонентно.

### 3. Автотесты (Двойной капкан)
Наши E2E тесты расширены. Playwright теперь должен валидировать не только кнопки UI, но и JSON-схемы ответов API, чтобы Contract-Driven подход имел строгий технический барьер перед релизом.
